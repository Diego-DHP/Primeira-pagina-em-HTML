Lógica do HTML

Para entender melhor o HTML nada mais é que uma

das estruturas principais da criação de um site,

dessa forma conseguimos fazer diversas coisas nele,

ele fica com umas das pricipais funções

relacionadas ao texto. Por exemplo, títulos,

cabeçalhos, textos de modo geral, organização e etc.




O que são Tags em HTML

As tag's no HTML são a forma que os textos vão se

comportar, imagine que você vai fazer uma lista

e você ter que pular uma linha colocar um asterisco

ou um número e pular pra outra linha e assim

por diante. Porem isso pode ser feito facilmente

ultilizando a tag "UL" para lista não ordenada,

ou seja vai ter uma lista com pontinhos antes do

item ou listada por números usando "OL".




Formatação em HTML

que você precisa saber é que, é sim possível

fazer inumeras formatações em HTML, porém

o foco principal dele não é este, é possivel

mudar a cor da fonte, marcação de texto,

entre outros. Porém é muito mais comum e

simples fazer estas alterações em CSS.
